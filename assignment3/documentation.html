GET Index

The GET method requests a representation of the specified resource. The GET Index requests a representation of the whole data in the web server.

Resource URL

http://localhost:3000

Resource Information

Response formats: JSON

Example Request

curl --location --request GET 'http://localhost:3000'


GET Show

The GET method requests a representation of the specified resource. The GET Show requests a representation of a specific item with a specific id paramater.

Resource URL

http://localhost:3000

Resource Information

Response formats: JSON

Parameters

NameRequiredDescriptionExample
idrequiredThe numeric value that represents the location from where to return trending information for from.2

Example Request

curl --location --request GET 'http://localhost:3000/2'


POST Create

The POST method submits an entity to the specified resource causing a change in state or side effects on the server. The POST Create submits data to the web server.

Resource URL

http://localhost:3000

Resource Information

Response formats: JSON

Example Request

curl --location --request POST 'http://localhost:3000' \ --data-raw '{ "os":"OSX", "brand": "Apple", "model" : "Iphone X", "image": "https://upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Fairphone_3_modules_on_display.jpg/320px-Fairphone_3_modules_on_display.jpg", "screensize":5.12 }'

Example Response

{ "os":"OSX", "brand": "Apple", "model" : "Iphone X", "image": "https://upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Fairphone_3_modules_on_display.jpg/320px-Fairphone_3_modules_on_display.jpg", "screensize":5.12 }
